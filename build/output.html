Technical Details

The transpiler follows a classic compiler pipeline: 1. <strong>Lexer</strong>: Scans the input and identifies tokens 2. <strong>Parser</strong>: Builds a structured representation 3. <strong>Generator</strong>: Converts the structure to HTML
